# Тестирование проекта на django
Сайт представляет собой прототип блога, где пользователи могут размещать свои публикации.

Тестовый код находится в приложении posts и users

posts:
Тестирование публикации постов с разными ролями
Содержание ответа на запросы
Работа кэша
Followers процессинг

users:
Регистрация нового пользователя
Проверка отсылки письма-подтверждения
Проверка созданной страницы профиля
