# Тестирование проекта на django
Сайт представляет собой прототип блога, где пользователи могут размещать свои публикации.

Тестовый код находится в приложении posts и users

posts:
1) Тестирование публикации постов с разными ролями
2) Содержание ответа на запросы
3) Работа кэша
4) Followers процессинг

users:
1) Регистрация нового пользователя
2) Проверка отсылки письма-подтверждения
3) Проверка созданной страницы профиля
